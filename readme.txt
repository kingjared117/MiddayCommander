Midday Commander
project submitted for Cs 3013 Project 1
Submitted by Jared Goldman and Oliver Simon
Submitted on 1/24/17
The files for the three different versions are stored in their own folders named v0, v1, and  v2 repectively.  
All files can be made using the correct make command.  make mc0 , make mc1 and make mc2 makes each of the respective versions.  make all will make all of the versions and make clean will remove the executables.
Examples are stored in each of the folders as well.  Examplecommands.txt will cointain a list of commands that are tested.  Exampletests cointains a captured output of the program using the commands shown in examplecommands.txt

Information on what the details of the project can be found in the project1.pdf
